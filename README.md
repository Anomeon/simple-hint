Simple Hint
===========

## Features
* positions (required)
	- top: `.hint-top`
	- bottom: `.hint-bottom`
	- left: `.hint-left`
	- right: `.hint-right`
* opacity fade
	- `.hint-fade`
* persist
	- `.hint-persist`
* sizing
	- small: `.hint-small`
	- medium: `.hint-medium`
	- big: `.hint-big`




### Known issues
* `.hint-fade` with `.hint-left` and `.hint-right` causes slight jump in text rendering
