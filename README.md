Simple Hint
===========
Clean CSS-only tooltips. 


## Features
- positioning
- opacity fade in
- animation
- delay visibility on hover
- persistence
- tooltip sizing

- Chrome, Firefox, Safari, Opera
- IE 9+ for basic usage
- IE 10+ for transitions & animations
